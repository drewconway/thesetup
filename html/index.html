Mapping Tool Communities from The Setup

This project is an attempt to map the communities of tools used by people interviewd by one of my favorite websites: The Setup. Each node on the map represents a tool used by an interviewee from The Setup. Node size is a proxy for each tool's overall popularity. The proximity of these tools to one another is a projection of their affiliation in interviews into a two-dimensional space. Put simply, the more people who use the same tools, the closer those tools will be to one another on the map. Communities are represented by color, and are determined hierarchically based on this proximity. Controls described below.

To learn more about the methodlogy used in this, and to download the code see my Github repository. This map will automatically update daily. Data may take a few seconds to load!

Controls

  • Zoom-in and -out using your mouse to view the node details
  • Increase or decrease the granularity of the communities using the right (→) and left (←) arrow keys, or by double-clicking on a cluster tick-mark.
  • Toggle map grid lines by pressing the "M" key
